Quiz Application

 Project Overview

The Quiz Application is an interactive web application developed using HTML5, CSS3, and JavaScript. It allows users to answer multiple-choice questions, automatically calculates the score, and displays the final result instantly.

 Features

- Interactive multiple-choice quiz
- Automatic score calculation
- Instant result display
- Clean and responsive user interface
- User-friendly design
- Hover effects on buttons

 Technologies Used

- HTML5
- CSS3
- JavaScript

 Project Structure

Quiz-Application/
│── quiz.html
└── README.md

 How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open quiz.html in any modern web browser (Chrome, Edge, Firefox, etc.).
4. Answer the quiz questions.
5. Click the Submit Quiz button to view your score.
